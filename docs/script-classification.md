# スクリプトの役割分類

## 概要

プロジェクト内のTypeScriptスクリプトを、その役割に応じて`bin`, `lib`, `src`の3つのカテゴリに分類する。

---

### `bin` （エントリーポイント / 実行スクリプト）
アプリケーションを直接起動したり、特定のタスクを実行したりするファイル群。

---

### `lib` （汎用ライブラリ / ユーティリティ）
特定の機能に特化し、複数の場所から再利用される部品となるファイル群。

---

### `src` （コアビジネスロジック）
エントリーポイントから呼び出され、アプリケーション固有の中核的な処理を担うファイル。
